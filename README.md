# 🎤 OS GOD - Voice Control System

A custom offline voice-controlled system to control your PC using commands.

## 🚀 Features
- Voice commands (Whisper-based)
- Modes (Zen, Alpha, Render)
- App automation
- Website automation
- Sound feedback system
- GG mode (instant sound trigger)
- System control (close apps, shutdown)

## 🧠 Example Commands
- "system activate zen mode"
- "system gg"
- "system close all"
- "system open notepad"

pip install -r requirements.txt
